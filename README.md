## 📌 개요
- 프로젝트 이름 : 동물상 미팅 서비스
- 서비스 운영기간 : 22.05.17 - 22.05.19 
- 개발 배경 : 세종대학교 2022축제 낮부스 운영
- 개발 언어 : JavaScript
- 프레임워크 : React
- 멤버 : 홍영환, 조준희
    
## 🖥️ 프로젝트 소개
 2022 세종대학교 축제 (05.17 - 05.19) 낮부스를 운영을 진행하였습니다.<br> 실제로 사용자가 이요할 수 있는 서비스를 제작해보고 수익창출 까지 경험해보고자 동물상미팅(AnimalFace Meeting)을 개발하게 되었습니다.

## ⚙️ 프로젝트 주요 기능
- 로그인
- 미팅참여 인원
- 매칭결과 확인
- 오픈채팅 참여
<br>
<br>
<strong>운영방법
1. 축제현장에서 자신의 동물상을 측정 후 등록
2. 이름과 전화번호로 웹사이트 로그인
3. 오후 6시에 매칭이 완료되면 매칭결과 확인
4. 오픈채팅방에 참여하여 미팅참가 완료!



## 🕰️ 개발 기간
- 22.05.04일 - 22.05.16일

## 👬 팀 소개
- 홍영환 - 백엔드(Firebase) , 유저매칭 웹(React) <br>
- 조준희 - 동물상 측정 웹(React), 동물상 모델 제작 (Teachable Machine)
